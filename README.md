# Study Cafe Web Application - Café Caloroso
### Node.js · Express · MongoDB · EJS

## Overview
This is a **full-stack, server-rendered web application** built using **Node.js, Express, MongoDB, and EJS**.  
The project provides an interactive interface for users to access pages such as home, login, menu, reservations, reviews, and contact forms.

The frontend is rendered using **EJS templates**, styled with CSS, and enhanced with client-side JavaScript, while the backend manages routing and database interactions.



## Tech Stack
- **Backend:** Node.js, Express.js  
- **Frontend:** EJS (Embedded JavaScript Templates), HTML, CSS, Vanilla JavaScript  
- **Database:** MongoDB (Local – MongoDB Compass)  


## Features
- User authentication pages (Login / Signup)
- Menu display
- Table reservation functionality
- Review submission
- Contact-us form


## Getting Started

### Prerequisites
- Node.js installed
- MongoDB running locally (MongoDB Compass)

### Installation
```bash
npm install
```
### Run the application
```bash
cd src
node index.js
```
The app will run on:
http://localhost:3000 (or the port defined in your configuration)
